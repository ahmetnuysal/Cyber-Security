# TCP ve UDP Farkları
* ### ```TCP``` daha düzgün veri aktarımında kullanılır (Bağlantı açma, web sitesine girme, mail alış-verişi)
* ### ```UDP``` daha hızlı veri aktarımında kullanılır (Video yayını)
